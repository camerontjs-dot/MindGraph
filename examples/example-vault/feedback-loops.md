---
title: Feedback loops
domain: systems-thinking
---

A feedback loop is a circular causal structure where the output of a process becomes part of its own input on a later pass. The two basic shapes are reinforcing loops, which amplify a change in the same direction, and balancing loops, which push the system back toward a setpoint.

Most real systems contain both kinds in interaction. A thermostat is a balancing loop on temperature. A viral product launch is a reinforcing loop on adoption. The interesting behavior usually comes from how these loops layer.
